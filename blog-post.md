# AI한테 시켜서 매일 아침 경제 뉴스를 이메일로 받아보는 시스템을 만들었습니다

매일 아침 출근하면 습관처럼 경제 뉴스를 훑어보게 되는데요.
네이버 뉴스 탭을 열고, 유튜브 경제 채널 몇 개를 돌려보고, 중요한 건 따로 메모하고...

그런데 이걸 매일 하다 보니 은근히 시간이 많이 들더라고요.

"이걸 자동으로 정리해서 매일 아침 메일로 보내주면 얼마나 좋을까?"

그래서 직접 만들어봤습니다. AI(Claude)의 도움을 받아서요.

---

## 먼저, 완성된 결과물을 보여드릴게요

매일 아침 9시, Gmail로 이런 메일이 도착합니다:

- 오늘의 경제 뉴스가 **주제별로 3건씩** 깔끔하게 정리되어 있고요
- 각 기사마다 **한줄 요약 + 원문 링크**가 함께 달려 있습니다
- 제가 구독하는 **유튜브 경제 채널의 오늘 영상**까지 같이 정리되어 있어요

클릭 한 번이면 원문으로 바로 이동할 수 있고, 관심 가는 기사만 골라 읽으면 됩니다.
블로그에 올리고 싶으시면 HTML 파일을 복사해서 붙여넣기만 하면 끝이에요.

---

## 어떻게 만들었을까요?

### 1단계: "이런 거 만들어줘"라고 말했습니다

Claude(AI 코딩 도구)에게 이렇게 요청했어요:

> "매일 경제 뉴스를 자동으로 수집해서 블로그 초안을 만들어주는 시스템을 만들어줘.
> 네이버 뉴스랑 유튜브에서 가져오고, 마크다운이랑 HTML로 저장해줘."

그랬더니 알아서 전체 구조를 설계하고, 필요한 파일들을 척척 만들어주더라고요.
프로젝트 구조, 코드, 설정 파일까지 전부요.

제가 한 건요? 앉아서 "응", "좋아", "진행해"를 누른 것뿐이었습니다.

---

### 2단계: API 키 발급 (유일하게 제가 직접 한 부분이에요)

자동으로 뉴스를 가져오려면 일종의 "출입증"이 필요합니다.
네이버와 구글에서 무료로 제공하는 API 키라는 걸 발급받았어요.

**네이버 검색 API** (뉴스를 가져오기 위한 용도)
- [네이버 개발자 센터](https://developers.naver.com/)에서 가입하고
- 애플리케이션을 등록한 뒤 "검색" API를 선택하면
- Client ID와 Client Secret이라는 두 개의 키를 받을 수 있습니다

**YouTube Data API v3** (유튜브 영상을 가져오기 위한 용도)
- [Google Cloud Console](https://console.cloud.google.com/)에서 프로젝트를 만들고
- YouTube Data API v3를 활성화한 뒤 API 키를 생성하면 됩니다

**Gmail 앱 비밀번호** (이메일을 자동으로 보내기 위한 용도)
- Google 계정에서 2단계 인증을 먼저 활성화하고
- [앱 비밀번호](https://myaccount.google.com/apppasswords)를 생성하면 됩니다

전부 무료이고요, 발급하는 데 5분이면 충분합니다.

---

### 3단계: 실행해보니 바로 결과가 나왔습니다

API 키를 넣고 실행했더니 이런 결과가 나왔어요:

```
=== 1. 뉴스 수집 ===
[INFO] 네이버 뉴스 147건 수집 완료
[INFO] 유튜브 영상 7건 수집 완료

=== 2. 포맷팅 (뉴스 147건, 영상 7건) ===

=== 3. 저장 ===
[SAVE] 마크다운 저장 완료
[SAVE] HTML 저장 완료

=== 4. 이메일 발송 ===
[EMAIL] 발송 완료
```

147건의 뉴스를 한꺼번에 가져왔는데요, 이걸 전부 보여주면 오히려 읽기가 힘들겠죠?
그래서 주제별로 3건씩만 골라서 보여주도록 설정했습니다.

**현재 수집하고 있는 주제는 이렇습니다:**
증시, 환율, 금리, 부동산, 조선주, 반도체, 재테크, 경제 일반

**유튜브 채널은요:**
한국경제TV, 슈카월드, 머니코믹스, 삼프로TV 등 5개 채널을 등록해두었습니다.

주제나 채널을 바꾸고 싶으시면 설정 파일 하나만 수정하면 돼요.

---

### 4단계: 처음엔 정보가 너무 많았습니다

사실 첫 번째 결과물은 뉴스 115건이 주르륵 나열되어 있었어요.
스크롤이 끝없이 내려가더라고요. 이건 정리가 아니라 정보 폭탄이었습니다.

그래서 Claude에게 이렇게 말했어요:

> "너무 정보가 많아. 주제별 3건씩만, 제목이랑 한줄 요약만 보여줘."

그랬더니 깔끔하게 압축된 버전으로 바로 바꿔주었습니다.

**수정 전:** 뉴스 115건, 끝없는 스크롤
**수정 후:** 주제별 3건, 한 화면에 쏙 들어오는 깔끔한 정리

이런 식으로 중간중간 피드백을 주면 AI가 바로바로 반영해주니까 정말 편했어요.

---

### 5단계: 매일 아침 자동으로 실행되도록 설정했습니다

매번 손으로 실행하면 그건 자동화가 아니잖아요.
GitHub Actions라는 기능을 활용해서 **매일 아침 9시에 알아서 돌아가도록** 만들었습니다.

작동 방식을 간단히 설명드리면요:

1. 매일 오전 9시(한국 시간)에 GitHub 서버가 자동으로 코드를 실행합니다
2. 네이버 뉴스와 유튜브에서 오늘의 경제 소식을 수집하고요
3. 주제별로 정리해서 블로그 초안을 만든 다음
4. 제 Gmail로 결과를 보내줍니다

이제 제가 할 일은 **아침에 메일함을 확인하는 것뿐**이에요.

---

### 6단계: 이메일이 안 와서 당황했습니다

GitHub에서 자동 실행을 시켰는데, 이메일이 오지 않았어요.
로그(실행 기록)를 확인해보니 Gmail 비밀번호가 서버에 잘못 저장된 거였습니다.

제 컴퓨터에서는 잘 되었는데, GitHub 서버로 옮기는 과정에서 값이 꼬였던 거예요.
비밀번호를 다시 설정하니까 바로 해결되었습니다.

개발할 때 이런 소소한 문제는 정말 흔하게 생기는데요,
중요한 건 **당황하지 않고 로그를 확인하는 것**이라는 걸 다시 한번 느꼈습니다.

---

## 다른 주제로 바꾸고 싶으시다면요?

설정 파일(`src/config.py`) 하나만 수정하시면 됩니다.

**뉴스 검색 키워드 바꾸기:**
```
현재: 증시, 환율, 금리, 부동산, 코스피, 코스닥, 경제, 조선주, 반도체, 재테크
```
여기서 원하시는 키워드를 추가하거나 빼시면 돼요.
예를 들어 "AI", "스타트업" 같은 키워드를 넣으면 해당 뉴스도 함께 수집됩니다.

**유튜브 채널 바꾸기:**
채널 ID를 추가하거나 삭제하시면 됩니다. 채널 ID는 유튜브 채널 페이지의 URL에서 확인하실 수 있어요.

**주제별 기사 수 조절:**
현재는 주제당 3건으로 설정되어 있는데요, 5건으로 늘리거나 1건으로 줄이는 것도 가능합니다.

## 비용이 궁금하시죠?

**0원입니다.**

- 네이버 검색 API: 무료 (하루 25,000건까지)
- YouTube API: 무료 (하루 10,000 유닛까지)
- GitHub Actions: Public 저장소는 무료
- Gmail 발송: 무료

모든 서비스가 무료 범위 안에서 충분히 돌아갑니다.

---

## 걸린 시간은요?

대화를 시작하고 완성하기까지 **약 30분** 정도 걸렸습니다.

그중에서 제가 실제로 한 일은요:
- API 키 발급하기 (약 5분)
- "응", "좋아", "진행해" 입력하기 (약 1분)
- 원하는 주제랑 유튜브 채널 알려주기 (약 1분)

나머지는 전부 AI가 해주었습니다.

## 마무리하며

솔직히 이 정도라면 개발자가 아니셔도 충분히 만들어보실 수 있다고 생각해요.
AI에게 "이런 거 만들어줘"라고 말하고, 중간중간 "이 부분은 이렇게 바꿔줘"라고 피드백만 주시면 됩니다.

하지만 코드를 직접 작성할 줄 모르셔도 괜찮습니다. 방향만 잡아주시면 AI가 알아서 구현해줍니다.
